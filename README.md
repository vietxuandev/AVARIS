# AVARIS Water - Premium Landing Page

A modern, premium landing page for AVARIS Water featuring glass bottle water services with stunning animations and responsive design.

## Features

- ⚡ Built with React + TypeScript + Vite
- 🎨 Styled with Tailwind CSS
- ✨ Smooth animations with Framer Motion
- 📱 Fully responsive design
- 🎯 Modern UI components with Shadcn/ui
- ♻️ Sustainable glass bottle focus

## Getting Started

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

### Preview

```bash
pnpm preview
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Shadcn/ui Components
- Lucide Icons
